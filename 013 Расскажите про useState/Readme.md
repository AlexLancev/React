### **Объяснение**

**useState** — хук для добавления состояния в функциональные компоненты.

```jsx
const [state, setState] = useState(initialValue);
```
- `state` — текущее значение
- `setState` — функция для обновления
- `initialValue` — начальное значение

---

### **Основное использование**

```jsx
function Counter() {
  const [count, setCount] = useState(0); // Начальное значение 0
  
  return (
    <div>
      <p>Вы кликнули {count} раз</p>
      <button onClick={() => setCount(count + 1)}>
        Кликнуть
      </button>
    </div>
  );
}
```

---

### **Особенности**

#### **1. Функциональное обновление**
```jsx
setCount(prevCount => prevCount + 1); // Лучше при множественных обновлениях
```

#### **2. Ленивая инициализация**
```jsx
const [state, setState] = useState(() => {
  // Выполнится только при первом рендере
  return computeExpensiveInitialValue();
});
```

#### **3. Объекты и массивы**
```jsx
const [user, setUser] = useState({ name: '', age: 0 });
// Обновление:
setUser(prev => ({ ...prev, name: 'Иван' }));
```

---

### **Резюме**

**useState** — хук для управления состоянием в функциональных компонентах.

**Ключевые нюансы:**
1. **Возвращает массив:** `[значение, функцияОбновления]`
2. **Асинхронность:** Обновления state могут быть асинхронными, несколько `setState` могут батчиться
3. **Иммутабельность:** Для объектов и массивов создавайте новые значения

**Как сказать своими словами:**
«useState — это способ добавить переменные-состояния в функциональный компонент. Возвращает пару: текущее значение и функцию для его изменения. При вызове функции обновления компонент перерисовывается с новым значением.»
